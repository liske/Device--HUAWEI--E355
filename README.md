Device::HUAWEI::E355
====================

About
-----

Device::HUAWEI::E355 is a perl package for easy access to the API calls
of the HUAWEI E355 high-speed packet access mobile hotspot.


Depends
-------

The following packages are required by Device::HUAWEI::E355:
* MIME::Base64
* POSIX
* XML::TreePP
* XML::Writer
* XML::Writer::Simple
